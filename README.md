# Astrological Signs
A mod that adds six new astrology themed items to The Binding of Isaac: Repentance.


## Features
- 6 Passive Items
- EID Suppport
- Encyclopedia Support

### Supported Languages
- **English** (Default)
- **Spanish** ((by Kotry) With EID)
